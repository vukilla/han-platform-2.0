# Security / Abuse / Privacy

## Upload security
- Rate limits on upload URL creation
- Virus/malware scanning on uploaded files
- Strict content-type validation

## Content rules
- Reject NSFW or unrelated content
- Require explicit consent for any human face data

## Privacy
- Store minimal PII
- Encrypt secrets and signed URLs
- Limit access to raw videos by role

## Abuse controls
- Throttle repeated failed uploads
- Track suspicious activity (rapid uploads, repeated invalid annotations)
